# Meridian

Binary Stock Outcome Markets on Blockchain — a Gauntlet weekly project.

Trade binary outcome contracts on whether MAG7 stocks close above a strike price today. Pays $1.00 USDC if yes, $0.00 if no. Non-custodial, on-chain order book, oracle-settled at 4:00 PM ET. Deployed to Solana devnet.
